CORE COMPONENTS:
  backend/              - FastAPI backend application
  frontend/             - React 18.2 dashboard
  keycloak/             - Keycloak authentication config
  docker-compose.yml    - Container orchestration
  *.py                  - Python scripts and utilities

DOCUMENTATION:
  LICENSE               - MIT License (legal)
  README_AUTHOR.md      - Complete project documentation
  AUTHORSHIP.md         - Ownership details
  OWNERSHIP_CERTIFICATE.md - Ownership certification
  COPYRIGHT_NOTICE.txt  - Copyright statement
  LEGAL_README.txt      - This file

DATA:
  10,488 stock records across 1,753 stocks
  Historical data from 2017-2024
  Sourced from SEC filings via Polygon.io

TECHNOLOGIES:
  Backend:    FastAPI, Python 3.11, SQLAlchemy
  Frontend:   React 18.2, Vite, JavaScript
  Database:   PostgreSQL 15, MongoDB 7
  Cache:      Redis 7
  Auth:       Keycloak 23.0 (OAuth2/OIDC)
  Infra:      Docker & Docker Compose
